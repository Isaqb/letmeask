***Iniciar****
yarn start
***Instalações*****
yarn add firebase
yarn add node-sass@^5.0.0
yarn add react-router-dom
yarn add @types/react-router-dom
yarn add classnames
***Arquivos Necessários****
node.js:versão 14.17.1
yarn:gerenciador de pacotes
firebase:usando auentifiação e database

**Hospedar**

hosting do firebase->npm install -g firebase-tools
$firebase login
->firebase logar com a conta cadastrada pelo firebase
$firebase init
->selecionar: realtime database,hosting
$yarn build
$firebase deploy
